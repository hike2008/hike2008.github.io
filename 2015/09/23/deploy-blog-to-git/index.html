#安装Hexo

hexo commands

1，进入Nodejs download下载对于版本的可执行文件，并安装。

2，执行命令npm instal hexo-cli -g。

此处可能会提示链接超时，此时建议使用淘宝镜像进行安装，具体命令如下：
`
  • npm install -g hexo-cli –registry=https://registry.npm.taobao.org`
    若不使用此镜像，则请自行翻墙安装。

3，hexo初始化。

新建一个文件夹，随便取名；右键-git bash打开git命令行；
执行命令：
hexo init blog;    //bolg为目标folder
cd blog;
npm install;

4，启动hexo服务器：hexo server。 //在对应目录打开命令行,执行如: E:/blog/node_modules/bin

5，在浏览器打开localhost:4000进行预览静态页面效果。

常用命令：
    初始化：  hexo init [folder]
    新建：  hexo new "Hello World" 
    编译：  hexo generate
            _Option    Description_
            -d,         --deploy        Deploy after generation finishes
            -w,          --watch        Watch file changes
    发布：     hexo publish [layout] <filename>
    启动服务器： hexo server
            Option    Description
            -p,     --port        Override default port
            -s,     --static        Only serve static files
            -l,     --log        Enable logger. Override logger format.
    部署：    hexo deploy
            Option    Description
            -g,         --generate    Generate before deployment

render
$ hexo render <file1[file2] …
Renders files.

Option Description
-o, –output Output destination
migrate
$ hexo migrate
Migrates content from other blog systems.

clean
$ hexo clean
Cleans the cache file (db.json) and generated files (public).

list
$ hexo list
Lists all routes.

version
$ hexo version
Displays version information.

Options
Safe mode
$ hexo –safe
Disables loading plugins and scripts. Try this if you encounter problems after installing a new plugin.

Debug mode
$ hexo –debug
Logs verbose messages to the terminal and to debug.log. Try this if you encounter any problems with Hexo. If you see errors, please raise a GitHub issue.

Silent mode
$ hexo –silent
Silences output to the terminal.

Customize config file path
$ hexo –config custom.yml
Uses a custom config file (instead of _config.yml).

Display drafts
$ hexo –draft
Displays draft posts (stored in the source/_drafts folder).

Customize CWD

$ hexo –cwd /path/to/cwd

6，部署博客到github。

  • 创建github仓库，名为username.github.io，建议初始化，建议同时初始化一个readme
  • 执行ssh-keygen -t rsa -C “user_eamil”(设置密码则push是需输入密码，否则可直接push)
  • 在github的settings —>SSH Keys—>ADD SSH Key 中将在C:\Users\Administrator.ssh目录中生成的id_rsa.pub中的内容拷贝到里面
  • 执行ssh -T git@github.com ， 建立远程连接
  • 执行npm install hexo-deployer-git –save ， 安装git部署插件
